== Description ==

A specific plugin use in Sksdev Theme to help you register post types.

Install Required Plugins: Sksdev Toolkit 

Activate plugin for more features and sections show in SKS Theme.

Contact Form show in frontend.

Show in dashboard menu
- Portfolio
- Team 
- Skills

Up coming version will have many more options

Found a bug or have a feature request ? Report here or have any question contact us at <a href="mailto:info@sksdev.com">info@sksdev.com</a>

== Installation ==

1. Put the "SKSDEV Toolkit.zip" extract to plugin folder "/wp-content/plugins/" or upload it.
2. Go into the WordPress admin pannel and activate the plugin
3. Enjoy
